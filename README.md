## 📖 使用说明

1. 点击上方导入链接，在 Loon 中确认导入。
2. 导入后即可在你的规则 / 脚本中使用图标名称。
3. 如果想添加更多图标，请参考项目目录结构：
   - `icons/`：存放图标文件（png）
   - `loon-icons.json`：图标集配置文件

---

## 📝 注意事项

- JSON 文件必须使用 **raw.githubusercontent.com** 的链接，才能被 Loon 识别。
- 仓库路径不要使用 `blob` 链接，否则导入会失败。
- 图标名称区分大小写，请确保在 Loon 内正确输入。
- 建议定期更新图标集，保证兼容最新 Loon 版本。
