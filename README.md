# 元气图标集 for Loon

欢迎使用 **元气图标集**！这是一个为 Loon iOS / macOS 用户准备的图标集合，包含常用应用、网站和主题图标，让你的 Loon 更加个性化。

---

## 📦 一键导入图标集

点击下面的链接即可在 Loon 中直接导入本图标集：

[一键导入元气图标集](loon://import?url=https://github.com/2Vrkswz/Loon-Icons/blob/main/iconset.json)

> ⚠️ 注意：
> - 仅在安装 Loon 的设备上有效  
> - JSON 文件必须是 raw.githubusercontent.com 链接  
> - Loon 版本需支持导入功能

## 📖 使用说明

1. 点击上方“一键导入”链接，Loon 会自动打开并导入图标集。  
2. 导入后即可在你的规则 / 脚本中使用图标名称。  
3. 图标集目录结构：
   - `icons/`：图标文件（PNG）  
   - `loon-icons.json`：图标集配置文件

---

## 📝 注意事项

- 确保使用的 JSON 链接是 raw.githubusercontent.com 格式  
- 图标名称区分大小写  
- 建议定期更新图标集，保持兼容最新 Loon 版本
